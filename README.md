# Medical-Image-Captioning-ResNet-LSTM
Deep learning models for medical image captioning using the ROCO dataset.

## Overview

This repository contains the implementation of a deep learning framework for medical image captioning using the ROCO dataset.

The repository includes the following models:

- ResNet50 + LSTM
- ResNet101 + Bahdanau Attention + LSTM

## Dataset

ROCO (Radiology Objects in Context) Dataset

## Requirements

Install the required packages using:

```bash
pip install -r requirements.txt
```

## Repository Contents

- ResNet50_LSTM.ipynb
- ResNet101_Bahdanau_Attention_LSTM.ipynb

## Evaluation

The models are evaluated using:

- BLEU
- ROUGE-L
- METEOR
